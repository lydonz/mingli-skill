# 紫微斗数方法论概要

> 基于陆斌兆《紫微斗数讲义》+ 王亭之注（中州派）+《紫微斗数全书》（明正统道藏版）
> 解法：以原典征验为准，不作主观引申

---

## 一、紫微斗数与子平八字的差异

| 维度 | 子平八字 | 紫微斗数 |
|------|---------|---------|
| 起源 | 唐李虚中 → 宋徐子平 | 陈抟传，宋代成书 |
| 推演基础 | 天干地支 + 五行 | 星曜 + 十二宫 |
| 时间维度 | 强（大运流年精到月） | 中（大运流年） |
| 空间维度 | 弱 | 强（十二宫覆盖人生全维度） |
| 性格分析 | 中（十神） | 强（主星辅星组合） |
| 事象分析 | 强 | 强 |
| 排盘复杂度 | 中（72 字左右） | 高（百余颗星） |

**两套系统不冲突，互相补充。** 八字看命运的"质"，紫微看命运的"量与位"。

## 二、排盘核心步骤

### 2.1 定命宫

由出生月支与时支推命宫地支（寅 = 1 起数）：

```
命宫地支 = 寅 + (生月数 - 生时数)
```

命宫天干用五虎遁（生年起寅顺数）。

### 2.2 定身宫

身宫与命宫同时定，根据出生时辰落入：命/夫妻/财帛/迁移/官禄/福德 六宫之一。

### 2.3 定五行局

由命宫纳音定（水二局/木三局/金四局/土五局/火六局）。**五行局决定起大运岁数和紫微星位置。**

### 2.4 安紫微

按五行局，从命宫起子，找紫微在哪一宫。然后顺安天机、逆隔太阳、顺安武曲...等。

### 2.5 安其他主星

按口诀依次安：天府（与紫微对照）、天机、太阳、武曲、天同、廉贞、太阴、贪狼、巨门、天相、天梁、七杀、破军。

### 2.6 安辅星煞星

- 左辅、右弼（生月起）
- 文昌、文曲（生时起）
- 天魁、天钺（生年起）
- 禄存、擎羊、陀罗（生年干起）
- 火星、铃星（生年支+时辰起）
- 地空、地劫（生时起）

### 2.7 起大运

按五行局对应岁数起，男阳女阴顺行，男阴女阳逆行。

## 四、十四主星核心特征（陆斌兆讲义）

### 4.1 紫微（北斗主，帝星）

- **本质**：尊贵、领导、独行
- **入庙**：午（极庙）、子、寅、申、辰、戌
- **关键格局**：
  - 紫微七杀同宫见化权 → "化杀为权" 上格
  - 紫府同宫 → 帝座加财库
  - 百官朝拱（左右昌曲魁钺禄马） → 真正发挥帝星之力
- **凶煞干扰**：见火铃羊陀 → 减格不破

### 4.2 武曲（财星 + 将星）

- **本质**：决断、刚强、财
- **关键格局**：
  - 武曲贪狼丑未 → 三十后发福，先贫后富
  - 武曲化忌 → 财星受克，结构性漏洞
  - 武曲会火铃 → 火铃贪格，主横发
- **最忌**：化忌 + 空劫同度

### 4.3 廉贞（次桃花，囚星）

- **本质**：决断冒险、感情、政事
- **特点**：陷地易凶（卯酉），庙旺（寅申）则吉
- **关键格局**：
  - 廉贞破军同宫 → 先破后立
  - 廉贞化禄 → 事业生财
  - 廉贞化忌 → 官非感情困扰

### 4.4 天府（南斗主，财库）

- **本质**：储藏、稳重、仁慈
- **入庙**：丑未、巳亥、卯酉
- **关键格局**：
  - 天府临迁移宫 → 出外得福（最佳位置之一）
  - 天府对紫微七杀（巳亥） → "最有威权的组合"
  - 必须见禄存或化禄才富

### 4.5 七杀（将星）

- **本质**：威猛、果决、孤剋
- **关键搭配**：
  - 七杀守命 → 喜禄存或化权制
  - 紫微七杀 → 化杀为权
  - 七杀加四煞 → 多刑剋

### 4.6 破军（先锋）

- **本质**：去旧换新、破坏、开创
- **关键搭配**：
  - 破军在卯酉 → 弃祖离宗
  - 破军化禄 → 后队补给，源源不绝

### 4.7 贪狼（次桃花，多才）

- **本质**：欲望、多才多艺、应变
- **关键格局**：
  - 贪狼庙旺 → 白手成家
  - 火贪铃贪格 → 横发横破
  - 武贪格（贪狼+武曲） → 三十后发福

### 4.8 巨门（暗星）

- **本质**：辩才、是非、深沉
- **关键搭配**：
  - 巨门化禄 → 因口舌得财
  - 巨门化忌 → 是非缠身
  - 太阳巨门同宫 → 异国发展利

### 4.9 太阳（中天主星）

- **本质**：光明、博爱、消耗
- **庙旺**：卯辰巳午（昼时区）
- **陷地**：戌亥子丑（夜时区）

### 4.10 太阴（中天主星）

- **本质**：内敛、柔和、积累
- **庙旺**：酉戌亥子（夜时区）

### 4.11 天机（智星）

- **本质**：机变、智慧、变动
- **关键搭配**：
  - 天机天梁同宫 → 必有高艺随身
  - 加羊陀空曜 → 易出家或漂泊

### 4.12 天梁（荫星）

- **本质**：清高、化解、贵气
- **化气**：化荫
- **特点**：救灾解难，但带孤独性

### 4.13 天同（福星）

- **本质**：温和、享福、被动
- **关键搭配**：
  - 天同太阴 → 需见吉星才发挥
  - 加杀重 → 肢体疾病

### 4.14 天相（印星）

- **本质**：辅佐、慈悲、随和
- **关键格局**：
  - "刑忌夹印"（前擎羊后化忌） → 大凶
  - "财荫夹印"（前化禄后天梁） → 大吉

## 五、四化（紫微斗数核心）

四化 = 化禄、化权、化科、化忌。

每个天干对应一组四化星。**大运、流年、流月按其天干起四化，飞入原盘十二宫。**

### 5.1 十干四化表

| 天干 | 化禄 | 化权 | 化科 | 化忌 |
|------|------|------|------|------|
| 甲 | 廉贞 | 破军 | 武曲 | 太阳 |
| 乙 | 天机 | 天梁 | 紫微 | 太阴 |
| 丙 | 天同 | 天机 | 文昌 | 廉贞 |
| 丁 | 太阴 | 天同 | 天机 | 巨门 |
| 戊 | 贪狼 | 太阴 | 右弼 | 天机 |
| 己 | 武曲 | 贪狼 | 天梁 | 文曲 |
| 庚 | 太阳 | 武曲 | 太阴 | 天同 |
| 辛 | 巨门 | 太阳 | 文曲 | 文昌 |
| 壬 | 天梁 | 紫微 | 左辅 | 武曲 |
| 癸 | 破军 | 巨门 | 太阴 | 贪狼 |

### 5.2 四化的含义

- **化禄**：财、缘、机会、流通
- **化权**：权、势、控制、主动
- **化科**：名、贵人、文书、清誉
- **化忌**：阻、损、纠缠、固执

### 5.3 飞星法判断

大运/流年的天干起四化 → 飞入原盘十二宫 → 看落到哪宫激活什么。

**示例：**
- 流年化禄飞入财帛宫 → 该年钱顺
- 流年化忌飞入夫妻宫 → 该年感情有阻
- 流年化禄+流年化忌都飞入同宫 → 该宫今年大震荡

### 5.4 化禄解化忌

原盘化忌 + 大运/流年化禄飞到同宫 → 化禄解化忌，凶中见吉。

## 六、关键格局诗诀（紫微斗数全书）

### 6.1 紫微在亥彖曰

> "亥紫到天門，孑身浮若雲。
> 憂不憂已事，識高必近君。"

四句意：紫微到亥宫为"天门位"，格局成立但孤独感强。心忧天下事不忧己事，认知水平高自然近权贵。

### 6.2 限到亥卯未

> "限到亥卯未宮，亦驟發驟富，
>  僧道清高，常人奮發。"

三方四正在亥卯未的人，运行至亥卯未宫位发福。

### 6.3 武贪丑未

> "武曲貪狼同丑未，少年不利，
>  三十後發福，先貧後富。"

三十岁前积累，三十岁后发福。早年莫强求。

### 6.4 武贪格的警告

> "武曲貪狼，見火鈴同度則主橫發，
>  再見刑忌，則橫發橫破。"

横发也容易横破，**得意不宜再往**。

### 6.5 天府临迁移

> "天府臨遷移宮，主出行得福，在外遇貴人。"

最佳出外发展星。

## 七、实战要点

### 7.1 看一个宫位看三方四正

任何宫位的判断不能只看本宫，必须看：
- 本宫
- 对宫
- 三合方两宫

四宫合看才有完整画面。

### 7.2 命宫定本质，身宫定取向

命宫 = 出生时定的本质（你是什么人）
身宫 = 后天追求的方向（你想成为什么人）

身宫落官禄 = 灵魂在事业里
身宫落夫妻 = 灵魂在感情里
身宫落财帛 = 灵魂在财富里
身宫落迁移 = 灵魂在远方
身宫落福德 = 灵魂在精神世界
身宫落命宫 = 自我认同极强

### 7.3 大运比流年重要

> "大運看支，歲君看干。"

大运是十年大背景，流年是一年具体表现。**大运好流年差只是小波折，大运差流年好只是回光。**

### 7.4 化忌的化解

化忌不是绝对凶。讲义提供五重化解路径：
1. 禄存照同宫 → 财路不断
2. 主星庙旺 → 跌倒能起
3. 化禄从三方来 → 对冲补血
4. 迁移宫吉 → 外部保护
5. 配偶宫吉 → 人事补救

操作原则：**赚到就走，不恋战；多线收入；异地理财；配偶守财。**

### 7.5 桃花星不在命/夫妻 = 非桃花命

天生桃花命的判断：贪狼/廉贞/天姚/咸池等桃花星落在命宫或夫妻宫。

桃花星散落其他宫位 = 桃花在某个具体维度（财帛宫桃花 = 因财遇异性）。

## 八、八字-紫微对照原则

### 8.1 八维度交叉

| 维度 | 八字看 | 紫微看 |
|------|-------|-------|
| 财运 | 财星状态 + 食伤生财链条 | 财帛宫主星 + 化禄化忌 |
| 事业 | 官杀 + 食伤 | 官禄宫 + 身宫 |
| 婚姻 | 日支 + 配偶星 | 夫妻宫 + 桃花星 |
| 迁移 | 驿马 + 冲 | 迁移宫主星 |
| 健康 | 五行受克 | 疾厄宫 |
| 贵人 | 天乙贵人 + 印星 | 天魁天钺 + 紫微百官朝拱 |
| 黄金期 | 大运喜用 | 大运走入吉宫 |
| 最差期 | 大运忌神 | 大运走入煞宫 |

### 8.2 矛盾时怎么办

两系统冲突时：
1. 重新核对排盘是否准确
2. 优先八字（时间精度高）+ 紫微辅证
3. 用已知事实验证哪边对
4. 标注分歧点，等后续事实校准

## 九、紫微斗数禁忌

1. **禁止只看主星**：辅星煞星同样重要，组合才决定吉凶
2. **禁止脱离三方四正**：单看本宫不准
3. **禁止忽略四化**：四化是动态变化的核心
4. **禁止机械套口诀**：每个盘的星曜组合都独特
5. **禁止简单粗暴定吉凶**：化忌不一定凶，化禄不一定吉
