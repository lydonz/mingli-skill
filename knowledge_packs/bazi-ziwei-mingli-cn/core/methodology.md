# 八字命理方法论

> 十五步断命流程，前后相续，缺一不可
> 认识论是"怎么看"的问题，见认识论.md
> 方法论是"怎么做"的问题

---

## 第一步：编码（排八字）

1. 确认出生年月日时（阳历），换算真太阳时（经度校正）
2. 排出四柱八字干支
3. 标出地支藏遁（人元），注明本气、中气、余气
4. 排出大运：阳男阴女顺行，阴男阳女逆行；从出生日数到下一个"节"（不是"气"），三日折一年
5. 确认起运年龄
6. 排出流年干支

---

## 第二步：识日主

1. 确认日干是十天干中的哪一个
2. 根据《滴天髓》十干论和《十干体象》诗，把握该天干的：
   - 喻象（甲=乔木，乙=花草，丙=太阳...）
   - 性质（刚/柔，动/静）
   - 喜忌要点（甲喜水润忌火焚，庚喜火炼忌土埋...）
3. 区分阳干阴干（阳从气不从势，阴从势无情义）

---

## 第三步：观月令

1. 确认月支对应什么月份、什么节气、什么季节
2. 查月令五行分日用事，确认当令之气是什么
3. 月令是整个分析的提纲——"取用凭于生月，当推究于深浅"
4. 月令地支的本气是什么？是否透干？
5. **注意季月（三月辰、六月未、九月戌、十二月丑）**：每季最后十八天土当令，须查具体生日在哪个分日用事区间
6. 陈素庵的纠正：取格以月支本气为主，不按"分日用事"的当值天干取格——"论命者必以本支为主，而后及其所藏也"

---

## 第四步：日主强弱分析

1. 标出八字中所有天元、地元、人元的五行
2. 分为日主同方（比肩劫财 + 正偏印）和日主异方（官杀 + 财星 + 食伤）
3. 结合月令时令，查五行四时旺相休囚死表，给各要素加权
4. 判断日主偏强、偏弱、还是中和
5. 注意：这只是一个视角的初步判断，不是最终结论

---

## 第五步：调候分析

1. 根据日干 + 出生月份，查《穷通宝鉴》十干配十二月调候用神
2. 看命局中是否有调候所需之神（透干？藏支？有根无根？）
3. 核心规则：
   - 冬寒需丙火暖局
   - 夏燥需癸水润局
   - "调候为急"——调候需求有时压过格局和强弱
4. 判断命局寒暖燥湿是否适宜

---

## 第六步：格局分析

### 6.1 取格
1. 从月令地支中取出格神——沈孝瞻："八字用神，专求月令"（注意：这里的"用神"是格神，不是扶抑用神）
2. 优先取月支本气，若本气不透或遭克，取月支中其他藏干
3. 若月支之神皆不可取，取"别干支之势盛力旺者为格"
4. 确认格神与日干的十神关系，定格局名称
5. 比肩劫财不取格（建禄格、羊刃格另论）
6. 西山易鉴先生首创六格法："曰官、曰印、曰财、曰杀、曰食神、曰伤官"——八格即六格分偏正

### 6.2 判断格局性质
- 善者（财、官、印、食）→ 顺用（生格 + 护格）
- 不善者（杀、伤、劫、刃）→ 逆用（制格）

### 6.3 找相神
- 善者格局：找能生格神的字（生格功能）+ 找能护格神的字（护格功能）
- 不善者格局：找能制约格神的字（制格功能）
- 不善者也可顺用（杀印相生、伤官生财等）

### 6.4 判断格局成败
- 成格条件：格神有力 + 相神配合得宜 + 无忌神破格
- 破格条件：忌神出现克破格神/相神
- 救应：有字能制住忌神（克制或合去），败中而成

### 6.5 评估格局高低
- 有情无情：配合是否得宜，结构要素是否和谐
- 有力无力：格神、相神是否通根有力

### 6.6 格局配气候
- "论命惟以月令用神为主，然亦须配合气候而互参之"
- 格局再好，调候不宜也打折扣

---

## 第七步：形象分析

从《滴天髓》角度，观察八字整体形象：

### 7.1 判断命局类型
- 独象（专旺格）：一行独旺
- 偶象（两神成象）：两行平分
- 从象（从格）：日主无根，满局克泄
- 化象（化气格）：天干五合化气
- 全象（正格）：三行以上

### 7.2 清浊
- 清：配置合宜，纯粹有精神
- 浊：官杀混杂、伤官见官、配置混乱

### 7.3 真假
- 真神：得时秉令，天透地藏，合于日主需要
- 假神：失时退气，勉强取用

### 7.4 源流
- 源头在哪里（最旺之气从哪里起）
- 流到哪里住（顺生到哪里停）
- 中间有无阻隔

### 7.5 通关
- 有阻隔的，看有无通关之神
- 通关之神是"一点精神所在"

---

## 第八步：确定全局用神和喜忌

综合以上所有视角，确定：
1. **用神**：最能帮助命局趋向中和/顺势的那个字
   - 日主强 → 以格局为主选用神（扶格或制格）
   - 日主弱 → 以扶身为主选用神（但不能破坏格局）
   - 调候为急时 → 调候用神优先
   - 有病 → 取药为用神
   - 有阻隔 → 取通关之神为用神
2. **喜神**：生助用神的
3. **忌神**：克破用神的
4. **仇神**：生助忌神的
5. **闲神**：与用神无直接关系的

注意：不同视角的用神可能冲突，须权衡取舍，不可机械套用。

---

## 第九步：论性情

1. 日干五行禀性（旺相 / 太过 / 不及）
2. 配什么五行多（金多 / 木多 / 水多 / 火多 / 土多）
3. 十神中哪个最突出：
   - 官重 → 循规蹈矩、有管理才能
   - 杀重 → 刚烈叛逆、有权威
   - 印重 → 智慧慈祥、重精神
   - 食重 → 温和宽厚、有口福
   - 伤重 → 才华横溢、傲物气高
   - 财重 → 务实勤劳、重物质
   - 比劫重 → 自我意识强、争竞
4. 综合描写命主个性特征

---

## 第十步：论疾病

1. 看五行受克受伤的情况
2. "取伤重者而断之"
3. 天干对应脏腑：甲胆乙肝丙小肠丁心戊胃己脾庚大肠辛肺壬膀胱癸肾
4. 天干对应部位：甲头乙项丙肩丁心戊胁己腹庚脐辛股壬胫癸足
5. 地支对应部位（查表）
6. 五行太旺太弱俱病

---

## 第十一步：论六亲

### 11.1 十神定六亲
- 男命：正印=母，偏财=父，正财=妻，七杀=子，正官=女
- 女命：正印=母，偏财=父，正官=夫，食神=子，伤官=女
- 注意：以用神为子、喜神为妻的原则具有更大概括性（《穷通宝鉴》《滴天髓》）

### 11.2 宫位定六亲
- 年柱 = 祖上
- 月柱 = 父母、兄弟
- 日支 = 配偶
- 时柱 = 子女

### 11.3 两者参合
- 看六亲星的旺衰、有无受伤、在哪个宫位
- 六亲星入其宫位最验
- 沈孝瞻："妻以配身，子为后嗣，亦是切身之事。至于祖宗兄弟，不甚验矣。"

---

## 第十二步：论财官（富贵贫贱）

1. 官星状态 → 社会地位、事业、权力
2. 财星状态 → 财富
3. 财官印食的配合情况
4. "专论财官"是今法时期论命核心
5. 现代延伸：官不限于做官，财不限于钱财

---

## 第十三步：大运流年动态分析

### 13.1 大运分析
1. 确定该步大运干支的五行属性
2. 通观干支，判断十年大方向
3. 看大运与原局的作用关系：
   - 与用神：生助？克泄？合绊？
   - 与忌神：制约？助长？
   - 与格局：助格？破格？救应？
   - 与日主：扶身？泄身？
   - 与调候：暖局？润局？
4. 看大运与原局的刑冲会合：
   - 冲提纲最重
   - 合走用神凶，合走忌神吉
   - 三合三会改变力量对比
5. 注意转角接木（方位转换）

### 13.2 流年分析
1. 流年天干代表表象，地支代表实质
2. 流年生助用神 = 吉；克破用神 = 凶
3. "日犯岁君，灾殃必重；五行有救，其年反必招财"
4. 岁运相生者吉；运克岁者凶
5. 岁运并临须看是否为忌神

### 13.3 断事象
- 看该年最受影响的十神 → 对应什么六亲、什么事类
- 官杀受冲 = 事业变动、官非
- 财星受冲 = 破财、父亲有事
- 印星受冲 = 母亲有事、学业变动
- 日支受冲 = 婚姻变动、迁移
- 驿马逢冲 = 远行迁徙
- 食伤旺年 = 才华发挥、表现机会

---

## 第十四步：神煞辅助验证

- 天乙贵人、天月二德、文昌、驿马、华盖、桃花、羊刃、空亡等
- 陈素庵："真者精择而存之，妄者悉举而削之"
- 神煞只是辅助信号，不能作为主要断法
- 在正五行分析结论基础上，神煞可以补充细节

---

## 第十五步：综合判断

1. 多视角分析结果的综合——不同视角的结论是否一致？矛盾处如何权衡？
2. **必须验证已知事实**："非稽其已往，无以测其将来"
3. 参以山川风土、门第世德、时代背景
4. 总评："浅而易见者小，清而难测者大，清而有神者贵，浊而无气者贱。纯粹中和者贵而安，奇怪偏驳者贵而危。"

## 禁忌

1. **禁止蒙猜**：不知道就说不知道，不要编造。"你他妈搁着蒙呢啊"——这是最严重的错误
2. **禁止反推**：不能拿已知事实反过来凑八字解释。"你不能拿事实推，因为前面这些这么大的事都看不到，后面的肯定都是瞎掰啊"
3. **禁止截看**：大运干支不可割裂为前后两段独立分析。任铁樵、陈素庵都明确反对
4. **禁止唯神煞论**：不能以一个神煞定吉凶。张神峰："此只把一字以定生死，此实谬说也"
5. **禁止唯格局论**：格局再好，调候不宜也打折。沈孝瞻："木逢冬水，虽透官星，亦难准贵"
6. **禁止唯用神论**：体不成则用无所附。陈素庵："苟日主六神或散漫无伦或战争不定，是则体先不成，用于何有"
7. **禁止机械套公式**：每个命都是独特的。"万物初生未成，而成久则灭。其超凡入圣之机，脱死回生之妙，不象而成，不形而化"
8. **禁止混淆格局用神和扶抑用神**：沈孝瞻的"用神"是格神，徐乐吾注释造成了至今的混淆
9. **禁止忽视十干性情差异**：同为木，甲乙完全不同。不能简化为"木就是木"

## 补充步骤（通读袁树珊韦千里后新增）

### 补充A：命宫推算（袁树珊体系）

**排法**：
1. 命宫地支 = 14 - 月支序数 - 时支序数（子=1，丑=2...亥=12；结果>12则减12，结果=0则取12=丑）
2. 命宫天干：用五虎遁法，从生年天干起寅，顺数至命宫地支

**用法**：
- 命宫补四柱之缺：四柱缺正财，命宫有正财 → 暗补
- 命宫藏忌神：命宫干支为忌神 → 暗伤
- 命宫参与刑冲合会：命宫地支与四柱互动
- 命宫可查神煞：命宫地支可作为贵人、文昌等的补充位置

**注意**：韦千里体系不用命宫。命宫是袁树珊的核心工具，属于古法子平（与紫微同源）。

### 补充B：小限推算（袁树珊体系）

**排法**：
1. 1岁小限 = 命宫干支
2. 逐年逆行（天干地支同时逆推）
3. 每12年地支循环一轮，天干持续逆行

**用法**：
- 十六字法：四柱8字 + 命宫2字 + 小限2字 + 大运2字 + 流年2字 = 16字
- 大运是十年背景，流年是当年主题，小限是个人这一年的际遇和感受
- 大运好+流年好+小限好 = 大吉；三者皆差 = 大凶；其他为混合

### 补充C：空亡分析

**排法**：
1. 从日柱往回数到甲X（旬头）
2. 该旬缺的两个地支即为空亡
3. 检查四柱地支中哪些落在空亡上

**判断**：
- 四柱中有地支落空亡 → 该地支上承载的十神、神煞受影响
- 空亡位上无四柱之字 → 不受影响
- 喜神空 → 吉力减半，忽有忽无
- 忌神空 → 凶力减半
- 双空（两个地支落空亡）→ 大起大落，波动极大
- "金空则鸣" → 金落空亡不是无金，是才华爆发型但不稳定
- 逢冲逢填之年 → 空亡被激活，可能爆发

**实战数据**（28人排查）：
- 有东西落空亡：大起大落型（蒋介石、张学良、邵逸夫、孙传芳、毛泽东等）
- 无东西落空亡：轨迹连贯型（张作霖、关羽、曾国藩、杜月笙、叶剑英等）

### 补充D：大运标注格式

每步大运应标注：步数、干支、年龄范围、年份范围、**干支十神**、**格局喜忌**、**调候喜忌**。

示例：
| 步 | 大运 | 年龄 | 年份 | 十神 | 格局 | 调候 |
|----|------|------|------|------|------|------|
| 2 | 庚戌 | 20-29 | 2022-2031 | 食神·比肩库 | 庚喜·戌忌 | 戌藏丁暖 |

### 补充E：调候与地理

八字是天的部分（时间），出生地/居住地是地的部分（空间）。五行方位：北方水、南方火、东方木、西方金、中央土。

- 八字缺火需调候 → 居南方天然补火
- 八字火旺需制 → 居北方天然补水
- 袁树珊案例：同一八字，生在南方得火气 vs 生在西方得金气，命运悬殊

**袁韦都有案例但没有系统化方法论。** 作为辅助参考，不作主要断法。

### 补充F：身强弱的正确看法（袁韦修正）

**食神泄秀的正确评估**：看身强弱时，看的是食神泄秀**之前**的状态。食神是药（强土得金方制其害），不是病。不应把食神算作"克泄耗"来削弱日主。

袁树珊千里命稿六神篇："旺土赖庚金食神以吐秀...土重为病，丙火为病中之病也。"——食神是解决"土重"这个病的药。

### 补充G：病药说实战（袁树珊体系）

袁树珊在命谱中对每一造都做病药分析。病分四类：
1. 刑冲破害（如寅申冲）
2. 调候不宜（如丙不透、水太寒）
3. 五行偏枯（如土太重、金太多）
4. 格局有瑕（如忌神透干）

药即解决病的那个字或那步运。"有病方为贵，无伤不是奇。"——先天有病有药才能成就非凡。
